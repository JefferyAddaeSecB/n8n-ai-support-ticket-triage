# n8n AI Support Ticket Triage

Workflow for AI-assisted support ticket classification, routing, and updates.

## Files
- `n8n/workflow.json` importable n8n workflow

## Import
1. Open n8n
2. `Workflows -> Import from File`
3. Select `n8n/workflow.json`
4. Update webhook paths, API URLs, and credentials
